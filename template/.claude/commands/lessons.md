학습 시스템 상태를 조회합니다.

## 절차

1. `docs/lessons-learned.md`를 읽어서 등록된 모든 L 엔트리를 요약 출력:
   - L번호, 제목, 위반 카운트
   - 관련 파일과 외부 시스템
   - 짧은 재발 방지 규칙

2. `docs/error-log.md`의 최근 10개 E 엔트리를 요약 출력:
   - E번호, 사용자 보고 (한 줄)
   - 수정 상태 (해결 / 미해결)
   - 승격 상태

3. 통계:
   - 총 등록된 L 수
   - 총 기록된 E 수
   - 최근 30일 신규 L 수
   - 가장 자주 위반되는 L (위반 카운트 기준 상위 3개)

## 사용자 명령 처리

이 슬래시 커맨드 다음에 사용자가 추가 명령을 줄 수 있습니다:
- "L3 자세히" → 해당 L 엔트리 전문 출력
- "E5 자세히" → 해당 E 엔트리 전문 출력
- "L3 취소" → error-curator에게 위임
- "이건 교훈이야: ..." → error-curator에게 명시적 등록 요청
